```
TVL / Total value locked / 总锁仓价值

在区块链的世界中，大部分 DeFi 活动需要引进抵押物，来确保整个生态的正常运转。这些锁定的抵押物的价值，就是 TVL 。 

TVL 的多少，一定程度上可以衡量一个项目的业务规模，以及生态发展状态。
```
